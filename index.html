El origen de HTML se remonta a 1980 cuando Tim Berners-Lee, un investigador del CERN, propuso un sistema de hipertexto para compartir documentos. Este sistema fue evolucionando y, junto con Robert Cailliau, desarrollaron el proyecto WorldWideWeb (W3), el cual introdujo el concepto de páginas web. En 1991, se publicó el primer documento formal sobre HTML, titulado "HTML Tags". En 1993, el IETF hizo la primera propuesta oficial para estandarizar HTML, y en 1995 publicó HTML 2.0, el primer estándar oficial. Desde 1996, el W3C asumió la responsabilidad de estandarizar HTML, publicando versiones como HTML 3.2 en 1997, HTML 4.0 en 1998, y HTML 4.01 en 1999. A partir del 2000, la atención del W3C se desplazó hacia XHTML, una versión basada en XML, mientras que HTML quedó en segundo plano. Sin embargo, en 2004, Apple, Mozilla y Opera crearon WHATWG, que impulsó la evolución de HTML hacia HTML5, cuya primera versión se publicó en 2008. En 2014, HTML5 se convirtió en un estándar oficial del W3C, promoviendo la creación de una Plataforma Web Abierta junto con tecnologías como JavaScript y CSS3. Esta nueva versión se centra en mejorar la seguridad, el diseño, la interactividad, la accesibilidad, y la usabilidad de las aplicaciones web modernas.

Su sintaxis en un elemento base al que llamamos marca, tag o simplemente etiqueta. A través de las etiquetas vamos definiendo los elementos del documento, como enlaces, párrafos, imágenes, etc. Así pues, un documento HTML estará constituido por texto y un conjunto de etiquetas para definir la función que juega cada contenido dentro de la página. Todo eso servirá al navegador para saber cómo se tendrá que presentar el texto y otros elementos en la página. Una etiqueta se define por que tiene una apertura: "<" y un cierre: "/>"

<img> Este es un texto de ejemplo de etiqueta

Las etiquetas de texto son aquellas que permiten introducir texto y modificar su formato de texto en html, es decir son las mas importantes porque dan la mayor parte del contenido de una pagina web. Ejemplos de estas etiquetas:

<p> Para parrafos

<h1> a <h6> son etiquetas de titulos y subtitulos

<span> Podemos personalizar el estilo de solamente una parte del texto

<strong> Se utiliza para destacar una parte del texto

<em> Se utiliza para emfatizar en un texto

<samp> Se utiliza para algun ejemplo

<cite> Se una para una cita

En HTML, podemos enumerar elementos en forma ordenada o desordenada. Una lista ordenada utiliza números o algún tipo de notación que indica una serie de elementos. Un ejemplo de su uso es:
<ol>
<li>Comer</li>
<li>Codigo</li>
<li>Dormir</li>
</ol>
El resultado seria:

  1. Comer
  2. Codigo
  3. Dormir
A esto tambien se le puede cambiar la numeracion de la lista utilizando el atributo "type" y se puden poner: A-a-I-i-1

Las listas desordenadas se basan en 2 tipos de etiquetas, la etiqueta <ul> para crearla y para el contenido de la lista la etiqueta <li>, aqui un ejemplo de correcto uso:
<ul>
<li>HTML</li>
<li>CSS</li>
<li>JS</li>
</ul>
El reultado seria:

  • HTML
  • CSS
  • JS
A todo lo antes mensionado se pueden poner distintos atributos, como poner encabezados con una etiqueta llamada nav, poner estilos a las litas e incluso poner la lista como hipervinculos.

Por ultimo se pueden crear listas con una breve descripción, esto se crea utlizando las siguientes etiquetas <dl> para crear la lista, <dt> para poner lo que vamos a describir y por ultimo <dd> para escribir la descripcion este tipo de lista la podemos crear de la siguiente manera:
<dl>
<dt>Codigo</dt>
<dd>Una actividad que me pone feliz, y luego me duermo</dd>
<dt>Caminar</dt>
<dd>Me sirve para distraerme</dd>
<dt>Dormir</dt>
<dd>Es la actividad que mas disfruto hacer</dd>
</dl>
El resultado seria:

codigo
Una actividad que me pone feliz, y luego me duermo
Caminar
Me sirve para distraerme
Dormir
Es la actividad que mas disfruto hacer

Las tablas HTML son elementos dentro del diseño web que permiten a los desarrolladores web organizar datos como texto, imágenes y enlaces en filas y columnas. Las tablas se crean de la siguiente manera: <table></table> Es para crear la tabla y de ahi se utilizan 2 etiquetas muy importantes las cuales son <tr></tr> para crear las filas de la tabla y los <td></td> para las celdas de datos, estas 3 son las mas comunes y utilizadas pero existen otras para encabezados, subtitulos y mas cosas :)

Ejemplo de la creacion de una tabla:
<table>
<tr>
<td>Hola, soy tu primera celda.</td>
<td>Soy tu segunda celda.</td>
</tr>
</table>
Y se puede seguir consecutivamente creando mas filas y con informacion diferente, el resultado es:

Hola, soy tu primera celda. Soy tu segunda celda. Soy tu tercer celda. Soy tu cuarta celda.
Hola Mundo

De igual manera se le puede aplicar distintos estilos a la tabla con CSS o con algun framework de CSS que tienen distintas propuestas de diseño

Para insertar una imagen en una página HTML se utiliza la etiqueta <img>. Es una etiqueta vacía que no contiene texto y que no necesita una etiqueta de cierre. Su atributo principal y el mínimo indispensable es el atributo src (por la palabra en inglés source) que, por medio de una URL, especifica el origen o la fuente de donde se obtendrá la imagen para ser mostrada en la página. Existen 2 formas de poner imagenes con HTML las cuales son tomando la URL de la imagen estando en la computadora o tomando la URL desde internet e insertarla a la pagina web.

Ejemplo de la etiqueta <img>:
<img hrf="Ruta de la imagen" alt="Es un mensaje que sale cuando la imagen no carga" width="Ancho de la imagen" height="Alto de la imagen">

Para insertar un video en una página HTML se utiliza la etiqueta <video></video>. En este caso es diferente a la de imagen porque esta si tiene un cierre de etiqueta eso quiere decir que entonces se pueden escribir fuentes del video, pero trabajan muy similares veamos sus atributos:

<video src="Url del video" controls width="Ancho del video" height="Alto del video" autoplay loop muted></video>

El atributo "controls" sirve para que puedan salir los controles del video, "autoplay" sirve para que el video se reproduzca automaticamente, "loop" sirve para que se reproduzca en bucle y "muted" para que este muteado el video veamos un ejemplo de como seria ejecutado:

La etiqueta <audio> en HTML es utilizada para reproducir audio en una página web. Es un elemento de bloque que permite mostrar un reproductor de audio en el navegador, permitiendo al usuario reproducir, detener, avanzar y retroceder el audio. veamos un ejemplo de su uso:

<audio src="Url del audio que quieras poner" controls autoplay loop ></audio>
Igual que con la etiqueta de video "controls" es para que aparezcan los controles del audio, "autoplay" para que se reproduzca automaticamente y "loop" es para que se reproduzca en bucle
